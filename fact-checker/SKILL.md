# 事实核查手册 🦞

> 写给龙虾，也写给每一个想把事实查清楚的人。

---

## 零、先记住三句话

1. **看到数字就警觉，看到引用就查原文，看到“目前”就问“现在还成立吗”**
2. **搜不到 ≠ 不存在，查不到出处 ≠ 编造的**
3. **你是帮作者加固，不是质疑作者**

---

## 一、什么时候该查？

读完一段话，问自己：**“这里有没有我愿意拿钱包打赌的数字或事实？”**

如果犹豫了——哪怕只犹豫了一秒——就标出来。

### 八个雷达


| 维度       | 触发词                  | 例子                        |
| -------- | -------------------- | ------------------------- |
| **人物**   | 人名、头衔、身份、关系          | “库克是乔布斯的继任者”              |
| **时间**   | 年份、日期、“目前”“现在”       | “《枪炮》1997年出版”             |
| **地点**   | 地名、方位、古今对应           | “君士坦丁堡即今天的伊斯坦布尔”          |
| **事物**   | 书名、机构名、带引号的引用        | “爱因斯坦说'想象力比知识更重要'”        |
| **数字**   | 百分比、金额、排名、对比         | “市场份额35%”                 |
| **时效性**  | “目前”“现行”“最新”“最大”     | “目前OpenAI的CEO是Sam Altman” |
| **绝对化**  | “100%”“都”“最”“所有”“公认” | “所有专家都认为”                 |
| **因果推断** | “因为A所以B”“导致”         | “喝咖啡导致成功”                 |


**前六个要搜，后两个标风险就行。**

---

## 二、怎么标？

用最简单的方式：

```
原文：新佳宜是长沙本地的便利店连锁，在湖南有上千家门店。
标注：✅ 门店数量 → 需查

原文：茶叶蛋一年大概卖出一亿个
标注：⚠️ 具体数字 → 需查，来源？

原文：华住真正花大钱的AI监控，是用来对付老鼠的
标注：🚩 比较性断言 → 需查华住AI投入分布
```

**原则：宁多勿漏。** 不确定要不要查？先标出来。

---

## 三、怎么搜？

### 关键词拆法

❌ 不要搜整句话：

> “新佳宜茶叶蛋一年卖出一亿个”

✅ 拆成“实体+属性”：

> “新佳宜 茶叶蛋 年销量”  
> “新佳宜 门店数量 2025”

### 搜索技巧


| 场景    | 怎么搜               |
| ----- | ----------------- |
| 查数字   | “品牌名 + 指标名 + 年份”  |
| 查引用   | “关键句 + 作者名”       |
| 查人物关系 | “人名A + 人名B + 关系词” |
| 查时效性  | 加时间限定，搜最新信息       |
| 查机构名  | 搜全称，看官网           |


### 搜到之后怎么判

看三件事：

1. **来源权威性**：政府官网 > 行业媒体 > 自媒体 > 没有来源
2. **时间**：2023年的数据不能支撑2025年的断言
3. **交叉验证**：至少两个独立来源一致才更可信

## 四、搜不到怎么办？

这是最容易犯错的地方。


| ❌ 禁止说法     | ✅ 正确说法          |
| ---------- | --------------- |
| “查无此据”     | “在公开渠道未找到原始出处”  |
| “无法证实”     | “现有搜索范围内未能验证”   |
| “没有任何证据支撑” | “建议补注来源，方便读者溯源” |
| “纯属编造”     | “无法从公开渠道核实”     |


**为什么？** 因为你的搜索工具有覆盖盲区。你搜不到的东西：

- 可能来自非公开渠道（内部演讲、私人交流）
- 可能发布在你看不到的平台（已下架、付费墙内）
- 可能是真的，只是没人写过

**搜不到 ≠ 不存在。** 你找不到出处，作者可能有你不了解的信息源。

## 五、结论怎么写？

### 铁律：先写 detail，再定 conclusion

**detail** = 我查到了什么  
**conclusion** = 所以结论是什么

这俩必须逻辑一致！

### 六种结论


| 结论                   | 含义   | 什么时候用         |
| -------------------- | ---- | ------------- |
| ✅ correct            | 正确   | 有权威来源支撑，与文章一致 |
| ❌ incorrect          | 错误   | 与事实不符，能给出正确信息 |
| ⚠️ partially_correct | 部分正确 | 有些地方有问题，说明哪部分 |
| 📅 outdated          | 已过时  | 曾经对，但现在变了     |
| ❓ unverifiable       | 无法核实 | 搜不到，但也不能说它错   |
| 🚩 risk              | 风险提示 | 绝对化表述或因果推断    |


### 正确示范

```
claim: 攀枝花五一温度骤降到17度
detail: 2025年五一攀枝花气温18-35℃，未出现骤降到17度的情况。
        数据来源：攀枝花官方五一专题天气预报
conclusion: ❌ incorrect
```

### 错误示范（绝对禁止！）

```
claim: GDP增长5.2%
detail: 实际为5.0%（来源：国家统计局）
conclusion: ✅ correct    ← 矛盾！detail说错了，conclusion却说正确
```

## 六、实战：拿这篇文章练手

下面是一篇真实的文章，我标出了关键核查点。你先自己查一遍，再对照答案。

### 练习文章（节选）

> 新佳宜是长沙本地的便利店连锁，在湖南有上千家门店。他们有一款产品是茶叶蛋，一年大概卖出一亿个左右。
>
> 华住连锁酒店真正花大钱的AI监控，是用来对付老鼠的。AI摄像头用的是红外检测，老鼠眼睛睁开、在那左摇右晃探路的动作，当天晚上就能识别到。
>
> 安徽有一家中式烘焙连锁叫张记糕点，是当地的中华老字号。西式烘焙店来来回去卖的就那22个核心产品；中式烘焙大概有137个SKU。
>
> 多点推行这套流程之后，生鲜损耗率从30%降到10%，生鲜板块的总利润上涨了30%。
>
> 攀枝花平时气温长期在30度，但五一那几天天气反常，温度骤降到了17度。

### 你的任务

1. 用八个雷达标出所有可疑点
2. 按风险排优先级
3. 搜一遍
4. 写结论（先 detail 再 conclusion）

## 七、核查红线（绝对不能犯的错）

- ❌ 搜不到就说“查无此据”“无法证实”“纯属编造”
- ❌ detail 写“实际是X”，conclusion 却填 correct
- ❌ 用户资料照单全收，能公网验证的也不验证
- ❌ 绝对化表述和因果推断不标风险就放过
- ❌ 把“无法核实”写成“有误”——这是两码事

---

## 八、最后一句话

核查不是跟作者作对，是帮文章变得经得起推敲。

你找到的每一个问题，都是在帮读者少一分质疑、帮作者多一分信任。

**干这行，友善的杠精比沉默的审计机器有用得多。** 🦞
